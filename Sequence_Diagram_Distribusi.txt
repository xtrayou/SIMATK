Title: Sequence Diagram - Distribusi & Pemotongan Stok ATK

# Aktor dan Komponen
actor Admin
participant View (Detail Permintaan)
participant PermintaanController
participant PermintaanModel
participant MutasiStokModel
participant BarangModel
database Database

# Alur Utama (Sequence)
Admin->View (Detail Permintaan): Klik Tombol "Distribusikan"
View (Detail Permintaan)->PermintaanController: HTTP POST (distribusikan)
PermintaanController->PermintaanModel: Cek Status Permintaan Saat Ini
PermintaanModel->Database: Query Status
Database-->PermintaanModel: Return Status (approved)
PermintaanModel-->PermintaanController: Validation Sukses

# Proses Update Permintaan
PermintaanController->PermintaanModel: Ubah Status = "distributed"
PermintaanModel->Database: UPDATE tabel requests
Database-->PermintaanModel: Return Success

# Looping Detail Barang (Pemotongan Stok)
PermintaanController->MutasiStokModel: Catat Mutasi Barang Keluar (OUT)
MutasiStokModel->Database: INSERT ke tabel stock_movements
Database-->MutasiStokModel: Return Success

PermintaanController->BarangModel: Kurangi Stok Saat Ini
BarangModel->Database: UPDATE barang SET current_stock = current_stock - Qty
Database-->BarangModel: Return Success

# Selesai
BarangModel-->PermintaanController: Loop Selesai (Stok & Mutasi Terupdate)
PermintaanController-->View (Detail Permintaan): Redirect dengan Noti Sukses
View (Detail Permintaan)-->Admin: Tampilkan Status "Didistribusikan"
